How To Contribute
=================

What To Do?
-----------
- Testing and finding bugs
- Fix bugs
- Implement new things
- Write documentation
- Donate

How To Contribute?
------------------
Your contribution can be of numerous forms. We currently accept two ways to contribute - Pull Requests and Patches.
